# Exámenes Anteriores ITCR - Archivos

Este repositorio almacena todos los archivos PDF y metadatos que alimentan la página web de [Exámenes Anteriores ITCR](https://github.com/rgdan/examenes-anteriores-itcr).

---

> [!WARNING]
> **Aviso de Discreción y Privacidad**  
> Respete siempre la discreción de los profesores. Evite publicar material de carácter confidencial, evaluaciones privadas o cualquier documento que no deba ser de acceso público.

---

## Estructura de este Repositorio

Aquí se organizan los exámenes en carpetas por escuela, materia y año, acompañados de sus respectivos archivos de metadatos.

## ¿Cómo contribuir?

Si desea agregar nuevos exámenes o corregir metadatos existentes, recuerde que **los archivos deben subirse a este repositorio (archivos)** y no al repositorio del código web.

Antes de realizar cualquier aporte, es **indispensable** que consulte la documentación oficial en la Wiki del repositorio principal:

* **[Guía de Contribución](https://github.com/rgdan/examenes-anteriores-itcr/wiki/Cómo-contribuir):** Pasos detallados para hacer un Fork, clonar, subir sus aportes y enviar un Pull Request.
* **[Nomenclatura y Convenciones](https://github.com/rgdan/examenes-anteriores-itcr/wiki/Nomenclatura-y-Convenciones):** Reglas para nombrar los archivos PDF y organizar los metadatos para que el script generador los procese correctamente.
